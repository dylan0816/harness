# 要点记忆与策略索引 (Point Memory Index)

> **AI 助手维护须知**：
>
> 1. 新增要点或策略演进后，必须同步更新此处的统计与关联。
> 2. 严格保持 Markdown 表格和列表的格式，严禁破坏原有结构。
> 3. 必须优先复用现有的 Level 1 和 Level 2 标签，避免标签碎片化。

---

## 新增要点格式

所有 reference 文件遵循统一格式：

```markdown
# 标题

> 适用场景：一句话说明在什么情况下参考本文件。

## 要点 N：标题

**现象**: 发生了什么。
**原因**: 为什么发生。
**解决**: 如何解决。
**经验**: 可复用的通用结论。

**标签**: `#tag1` `#tag2`
**来源**: 实践踩坑 | 代码审查
```

**规则**：

- 现象/原因/解决/经验 四段完整，缺一个说明原因
- 标签从现有 Level 1/2 中优先复用，新增标签需同步更新本索引
